# Microservices Tooling

----

## Install Weave Scope

```bash
sudo wget -O /usr/local/bin/scope https://git.io/scope
sudo chmod a+x /usr/local/bin/scope
sudo scope launch
```

----

## Open Scope UI

`http://[Cloud VM IP]:4040/`

----

## Try it out!

- View metrics
- Kill container (watch results)
- Connect to container
